## 백엔드 Python Huggingface API 활용 모델


### 레포지토리 저장소 내려받기
```zsh
$ git clone https://github.com/7eerup/backend.git
```

### 가상 환경 설정
```zsh
$ python -m venv .venv
$ source .venv/bin/activate
```

### 패키지 설치
```zsh
$ pip install -r requirements.txt
```
